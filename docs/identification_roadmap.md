> "只有公开数据 + 已接 Tushare"的现实约束下，哪几条路真正可行、各自的

## 0. 一句话结论

当前论文是一篇**工程扎实、口径诚实的描述性跨市场事件研究**；它的天花板是
**没有任何一个环节构成因果识别**——主线（announce vs effective 事件研究）是
描述性时序分解，唯一的准实验设计（HS300 RDD）在现有公开数据下不成立。
要突破天花板，**真正的杠杆是"识别"，不是再润色数字或图表**。但在"只有公开
数据"的硬约束下，能走的路有限且各有代价；本文档把它们一一摆清。

## 1. 为什么当前不构成因果识别（精确诊断）

### 1.1 主线：announce-vs-effective 事件研究 = 描述性，不是因果

- 主结论"公告窗 CAR 显著为正、生效窗约为零、中美量级相近"是一组**事实**，
  方法上无懈可击，但它**不识别任何因果参数**：
  - 它比较的是同一批纳入事件在两个时间窗的平均异常收益，没有反事实
    （"如果不纳入会怎样"）；
  - "跨制度相似性即信息渠道证据"是一个**论证**，不是识别——它依赖
    "若需求渠道主导则中美应分化"这一辅助假设，而该假设无法被检验
    （被动 AUM 口径、市值口径、样本期都不可比）；
  - 论文自己已诚实写明："描述性识别，非准实验；生效窗约为零也与需求被
    提前套利相容"（`docs/paper_outline.md` §3.2）。这句话本身就是
    天花板的自我声明。

### 1.2 唯一的准实验设计（HS300 RDD）当前不成立

两个独立的硬伤，任一都足以让审稿人否掉因果声明：

1. **running variable 是伪造的排序序数，不是连续经济变量。**
   `analysis/rdd_reconstruction.py` 把 running variable 构造成
   `(2·cutoff+1) − descending_rank`，即"调整名单顺序"映射成边界序数——
   它与处理变量 100% 共线，RDD 的识别假设（个体无法精确操纵跨越断点的
   连续得分）根本无法陈述。主设定 τ≈3.0%（p=0.095）是"调入组 vs 擦肩而过组"
   的均值差，不是断点处的因果跳跃。
   - 注意：项目已把 L3 升级为"中证官方候选名单 + 备选名单排序列"
     （`hs300_rdd_l3_collection_audit.md` §7 2026-05-03 记录），比纯反推
     好，但**仍是"官方调整名单序"的 boundary ordinal proxy，不是中证内部
     真实 ranking score**。后者不公开发布。

2. **效应不稳健：一阶仅边际显著，二阶多项式直接塌缩到零。**
   `results/literature/hs300_rdd/rdd_robustness.csv` 实测（Tushare 口径，
   outcome=`car_m1_p1`, bandwidth=0.06）：
   | 设定 | τ | p | n_obs |
   |---|---|---|---|
   | main · local linear (order 1) | **+0.0296** | **0.095** | 148 |
   | donut(±0.01) | +0.0384 | 0.165 | 124 |
   | polynomial order 2 | **−0.0025** | **0.939** | 148 |
   | placebo cutoff −0.05 | −0.0063 | 0.770 | 84 |
   | placebo cutoff +0.05 | −0.0200 | 0.116 | 174 |
   主设定本身只到 **p=0.095（连 0.05 都没过）**，donut 后 p=0.165 失去显著，
   二阶多项式 τ 翻负、p=0.94——这是"断点是函数形式假象 + 效应脆弱"的双重
   信号。审稿人第一刀就会砍这里。
   （注：placebo 在假断点处不显著，是唯一一项behave-as-expected 的检验，
   但救不回主设定的脆弱性。）

3. **样本不足。** L3 = 11 批次 / 5 年（2020-11 至 2025-11，356 候选）。
   论文级 RDD 通常要 ≥10 年 / ≥20 批次。

### 1.3 H1–H7 机制假说是 post-hoc，且 Tushare 后更脆

- 7 条假说在**观察到结果之后**形成，无预分析计划（已诚实归入附录）。
- 换 Tushare 数据后 **H5、H2 翻转为"证据不足"**
  （`docs/yahoo_vs_tushare_comparison.md`）——这反而是诚信加分项，但也
  坐实了"这些不是稳健的确证结论"。机制层不能作为因果主张的支柱。

## 2. 现实约束（决定哪些路是死的）

| 约束 | 后果 |
|---|---|
| **只有公开数据，无 Wind / CSMAR / iFinD** | 中证 2020 前不公开备选名单（`hs300_rdd_l3_collection_audit.md` §5 已用 13 个搜索词穷尽验证）→ **RDD 扩到 10 年这条路在现有数据下基本死亡**，除非买数据库或啃 Wayback 档案（命中率低且附件仍不含 reserve list）。 |
| **已接 Tushare Pro（A股持牌）** | A 股价格/成交量/市值/换手率/涨跌停字段现在可信 → 任何依赖 A 股**微观结构**的识别（涨跌停断点、流动性、买卖盘）数据基础变扎实了。 |
| **US 端仍是 Yahoo 近似** | 美股自由流通市值/成分权重是近似 → 跨市场"权重 × 被动持仓"类识别在 US 端不可靠。 |
| **指数纳入文献已非常成熟** | "纳入有没有效应"是 1986 年的老问题；新贡献必须在**机制/制度/识别**上，不能停在"效应存在"。 |

## 3. 候选路径（按可行性 × 价值排序）

下面每条标注：数据是否够、工作量、对 PhD/量化的价值、死路在哪。

### 路径 A —— S&P 500 float-adjustment 权重断点（★ 最推荐做可行性探查）

**思路**：放弃 HS300 RDD，改用**美股**真正的准实验——S&P 500 的纳入由
委员会决定（有自由裁量，不是纯阈值），但**纳入后的指数权重**由 float-adjusted
市值决定，存在文献中成熟的识别变体（如 Chang-Hong-Liskovich 2014 用
Russell 1000/2000 断点；S&P 也有 free-float band 调整）。

- **数据**：S&P 500 历史成分 + float-adjusted 权重。Yahoo 不够，但
  **部分可从公开来源拼**（S&P 官方方法论 + 公开 13F 被动持仓 + Wikipedia
  变更表已有）。需要核实能否拿到 float weight 历史。
- **工作量**：中（数据采集 2-4 周）。
- **PhD 价值**：★★★（一个真 RDD/IV 设计，哪怕初步，远胜描述性）。
- **量化价值**：★★（权重断点 = 可交易的 rebalance 信号，私募会感兴趣）。
- **死路风险**：float weight 历史可能同样要付费数据；需先做**1 天可行性
  探查**确认数据可得，再投入。

### 路径 B —— 涨跌停作为 A 股特有的识别杠杆（★ Tushare 红利最大化）

**思路**：H5（涨跌停截断）虽然在均值回归下翻成"证据不足"，但涨跌停本身
是 A 股独有的、**外生的价格上限机制**——这是中国市场相对美国的真正
制度差异点。换 Tushare 后涨跌停命中数据可信了。可以把它从"post-hoc 机制
回归"重做成一个**更接近识别的设计**：比较"公告日触及涨跌停 vs 接近但未触及"
的后续价格发现速度（bunching / 截断回归思路）。

- **数据**：⚠️ 修正——存盘的 `real_prices.csv` 只有 `close/ret/volume/turnover/mkt_cap`，
  **没有保留 `pct_chg` / high / low / 涨跌停价 / 封板标志**。涨跌停命中只能用
  `|ret|≈0.10` 近似（这也正是当前 H5 的口径）。严格的 bunching/截断要重抓
  Tushare `daily` 的 pct_chg + limit 字段并区分 10/20/30% 板。
- **工作量**：中（重做设计 + 写作，2-3 周）。
- **PhD 价值**：★★（制度识别有新意，但 bunching 设计本身也要小心操纵性检验）。
- **量化价值**：★★★（涨跌停后的价格发现 = 直接的 A 股交易信号）。

**⛔ 可行性探查结果（2026-05-30，已用真实面板数据跑过 — 路径 B 基本不可行）**：

用 `data/processed/real_event_panel.csv`（CN 纳入事件 137 个）按 `|ret|≥9.5%`
近似涨跌停命中，实测触板密度（数字均为真实跑出，非估计）：

| 阶段 / 窗口 | 触板事件数 | 占比 |
|---|---|---|
| 公告日 (day 0) | **1 / 137** | 0.7%（**唯一一个是涨停，0 跌停**）|
| 公告 [-1,+1] | 7 / 137 | 5% |
| 公告 [0,+5] | 5 / 137 | 4% |
| 生效日 (day 0) | 3 / 137 | 2% |
| 生效 [0,+5] | 8 / 137 | 6% |

公告日 ret 分布**最大值仅 +0.099**、+0.10 处只有 1 个、−0.10 处 0 个——
在断点附近**没有任何堆积（bunching）**，反而平滑稀疏到边界。

**结论**：bunching/截断回归需要断点附近有足够密度的样本，而公告日只有
**n=1**（且单边涨停、无跌停侧可对比；放宽到 [-1,+1] 也只有 7 个）——任何断点设计都是个位数样本上的噪音，
统计上无意义。这恰好解释了**为什么 H5 换 Tushare 后翻成"证据不足"**：不是
数据精度问题，而是**公告日涨跌停截断这个现象本身就很弱**（96% 的纳入公告
价格反应温和、远不到板）。即使放宽到生效后 [0,+5]（27 个），也已偏离
"纳入效应"主线、且仍受 ret 近似限制。**路径 B 在公告口径下死路，不建议投入。**

### 路径 C —— 诚实重定位为"描述性跨市场事实 + 消失的指数效应复制"

**思路**：不追因果，把论文**诚实地**定位成"对 Greenwood-Sammon (2022)
'消失的指数效应'的跨市场复制 + 中国首次系统的 announce/effective 分解"。
贡献从"识别因果"降级为"高质量描述性事实 + 文献复制 + 诚实的稳健性"。

- **数据**：现有 Tushare 口径**完全够**，不需新数据。
- **工作量**：低（重写摘要/引言/贡献句 + 强化稳健性表述，1 周）。
- **PhD 价值**：★（描述性论文进不了好期刊，但作为 writing sample 证明
  "懂识别的边界、不吹牛"反而是成熟度信号）。
- **量化价值**：★★（工程 + 诚实 = 私募看重的特质，配合代码仓库很强）。
- **死路风险**：无。这是**保底路径**——即使 A/B 都探查失败，C 一定能交付。

### 路径 D —— 把 HS300 RDD 扩到 10 年（⚠ 现有数据下基本是死路）

**思路**：补 2014-2019 的中证官方备选名单，让 RDD 够 10 年。

- **数据**：`hs300_rdd_l3_collection_audit.md` §5 已**实证证明**中证 API
  在 2020 前不发布备选名单；Wayback 命中的也是不含 reserve list 的网页；
  唯一可靠来源是 **Wind/iFinD/CSMAR 付费数据库**（你没有）。
- **结论**：**除非买数据库，否则这条路走不通。** 不建议投入，除非你能
  通过学校图书馆拿到 CSMAR/Wind 访问权（很多高校有，值得确认一次）。

## 4. 两个目标的不同取舍

| | PhD 申请（新加坡） | 量化私募求职 |
|---|---|---|
| **委员会/HR 真正看** | 研究品味、识别严谨度、能独立完成完整研究 | 工程能力、数据处理、信号可交易性、诚实度 |
| **最优路径** | A（真识别设计）> B > C | B（可交易信号）≈ C（工程+诚实）> A |
| **底线** | 至少 C 的诚实重定位 + 明确写出识别局限（成熟度信号） | 现有仓库 + C 已经是很强的 portfolio |
| **致命点** | 把描述性包装成因果（被一眼看穿 = 减分） | 信号不可交易 / 结论站不住 |

**关键洞察**：两个目标都**不要求**你现在就攻下因果识别。
- PhD：一份诚实、自知边界、工程极强的 writing sample，比一篇硬吹因果、
  被导师一眼看穿的论文更有利。
- 量化：仓库本身（48 CLI、全自动复现、Tushare 接入、诚实的稳健性追踪）
  已经是顶级 portfolio，C 路径足以收口。

## 5. 建议的下一步（最小承诺、先探后投）

按"先花最少时间确认数据，再决定投不投"的顺序：

1. **半天～1 天可行性探查**（不写论文，只验数据）：
   - 路径 A：能否拿到 S&P 500 历史 float-adjusted 权重（公开来源）？
   - 路径 B：用 Tushare 重抓"公告日涨跌停命中"样本，看截断/bunching
     设计下效应方向是否稳定？
   - 路径 D：问一次学校图书馆是否有 CSMAR/Wind 访问权（一封邮件的成本）。
2. **探查结果出来后**再决定主攻 A 还是 B；**无论 A/B 成败，都先把 C
   的诚实重定位做掉**（它是保底交付，且能立即提升材料质量）。
3. 任何路径都**不要**回头去硬救 HS300 RDD 的因果声明——把它永久定位为
   附录里的 illustrative/preliminary，并在正文一句话讲清它为什么不进主表。

## 6. 一句话给 Leo

天花板是真的，但它**不挡你当前的两个目标**。最高杠杆动作不是"再修数字"，
而是：①花 1 天探 A/B 数据可行性 → ②无论如何先做 C 的诚实重定位。
不要为了"看起来像因果"去硬撑一个站不住的 RDD——对 PhD 和量化，
**诚实地知道边界**比假装越过边界更值钱。
