## 1. 背景与动机

本项目原以"跨市场不对称(CMA)"为主线,围绕 7 条机制假说(H1–H7)+ HS300 RDD
组织论文。2026-05-22 的完整评估发现该框架站不住:

- **七假说是 post-hoc 拟合**,部分裁决建立在极小样本上(H3 n=4;H5 涨跌停在
  118 个 CN 公告事件中仅 3 个逼近 ±10% 板,其 p=0.008 回归 R²=0.011,系数实由
  那 3 个事件撑着)。
- **HS300 RDD 的 running variable 是伪造的**——它是调整名单序号
  (`(2·cutoff+1) − rank`),与处理变量 100% 共线,不构成有效识别。
- **核心的"中美不对称"基本不存在**:公告 CAR[-1,+1] 中国 +1.76%、美国 +1.84%,
  生效窗两边都 ≈ 0——两个市场几乎一样。

因此本文放弃"解释一个并不存在的不对称",改为提出一个数据真正支持、且诚实的论点。

## 2. 论点与研究问题

**论点**:指数纳入的正向价格效应是公告 / 信息效应,不是需求 / 价格压力效应——
两个制度极不相同的市场给出几乎相同的答案。

**研究问题**:指数纳入溢价来自 (a) 信息认证(被纳入是对公司质量的背书,公告即被
定价),还是 (b) 被动 / 指数基金的机械买盘(向下倾斜的需求曲线)?

## 3. 识别策略

利用每个事件的两个日期 × 两个市场:

- **公告日**:信息进入市场,此时尚无机械的指数基金调仓。
- **生效日**:指数基金必须机械买入,但此时无新信息(生效日提前公告、完全可预期)。
- **美国 S&P 500**:被动资金占比高、机构主导。
- **中国 CSI 300**:散户主导、被动资金占比低、制度差异大。

预测:
- 需求假说 → 效应集中在**生效窗**、且**美国 > 中国**。
- 信息假说 → 效应集中在**公告窗**、且**中美相近**。

这是经典 Harris-Gurel (1986) 时点分解逻辑的跨市场扩展。属于观察性识别,非准实验。

## 4. 核心实证结果

数字来源:`results/real_tables/event_study_summary.csv`(已核实,inclusion=1)。

| 窗口 | 中国 CSI 300 (n=117) | 美国 S&P 500 (n=254) |
|---|---|---|
| 公告 CAR[-1,+1] | +1.76% (t=4.93, p<0.001) | +1.84% (t=5.25, p<0.001) |
| 生效 CAR[-1,+1] | +0.42% (t=0.93, 不显著) | −0.14% (t=−0.51, 不显著) |

判读:效应集中在公告窗、生效窗 ≈ 0、且跨制度量级相近 ⇒ 指向信息渠道。
"跨制度相似性"是核心证据:中国(散户主导、被动少)本应在需求假说下表现不同,
实际与美国一致。

**诚实限定**:生效窗 ≈ 0 亦与"需求被提前预期 / 套利掉"(Greenwood-Sammon 2022)
相容,故单一窗口不能定论;论证依赖**跨市场相似 + 公告窗集中 + 长窗口不回吐**
三者合一。文中须并列这两种解释。

## 7. 诚实写明的弱点(论文 §7 与 `docs/limitations.md` 须包含)
- 描述性事件研究 + 时点分解,非准实验识别。
- 论点接近现代共识(信息渠道、需求效应消退);增量在系统的中美对照与
  "相似性即证据"的明确表述。
- CN 样本仅 ~117 可用事件 / 5 年(2020–2025);US 619 事件 / 2010–2025。
- mkt_cap / turnover 来自 Yahoo `sharesOutstanding` 近似,非交易所历史口径。
- 生效窗 ≈ 0 的多重解释(真无效应 vs 需求被套利)在文中须并列,不可只取其一。
